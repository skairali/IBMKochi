# IBMKochi

June 8th, 2022: IBM  announced the launch of a new  Innovation Centre, which is housed within the IBM India Software Lab in Kochi, located at Brigade World Trade Centre Info Park, Kakkanad. The new centre, which became fully operational in Q3 2022
